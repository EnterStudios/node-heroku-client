# organization-payment-method

The on file payment method for an account

## Actions

Method | Path
--- | ---

### `get`

`heroku.organizations({organization_name}).paymentMethod().get({callback});`

Method | Path
--- | ---
GET | /organizations/{organization_name}/payment-method
